# 데이터셋 전역 랜드마크 이름/약어 + 디지타이징 보조(Show Expected)

## 날짜
2026-07-22

## 배경

곡선/세미랜드마크 작업과 같은 날, 디지타이징 화면의 **라벨링과 예측 보조**를
개선했다. 두 축: (1) 랜드마크를 인덱스 대신 사람이 붙인 이름/약어로 식별,
(2) 미완성 표본을 찍는 동안 남은 랜드마크의 예상 위치를 미리 보여주기.

## 작업 내용

### 데이터셋 전역 랜드마크 이름/약어
- `MdObject`가 아니라 데이터셋 차원에서, 각 랜드마크 인덱스에 **이름/약어 +
  설명**을 부여. `MdDataset.landmark_name_json`(JSON, null 허용)에 저장,
  접근자 `get_landmark_names()`/`set_landmark_names()` — 끝쪽 빈 항목은 trim.
- 뷰어에서 인덱스 대신 이름/약어를 표시(예: `CR1`), 설명은 툴팁.
- 랜드마크 라벨 UI: 라벨을 **Show 체크박스 + Index/Name 라디오**로 바꿔 표시 방식
  선택, 설명 툴팁 제공.

### 디지타이징 보조 — Show Expected (2D)
- `cbxShowExpected` 체크박스. 새(미완성) 2D 표본에 **랜드마크를 2개 이상** 찍으면,
  데이터셋 평균 형상을 그 점들에 맞춰(fit) 정렬하고, **남은 위치를 정렬된 평균에서
  가져와** 예상 위치로 뷰어에 표시. 결측 랜드마크 임퓨테이션과 동일한 경로
  (`impute_missing_landmarks`)를 재사용.
- 뷰어는 아직 안 찍은(placed 이후) 항목만 그린다. 보조가 꺼져 있거나 계산 불가면
  `None`.
- 후속 수정: 예측 참조가 **더 긴(랜드마크 수가 많은) 표본에만 있는** 랜드마크도
  예측하도록 보정.

### 오브젝트 목록/미리보기 소소한 개선
- 오브젝트 목록의 "Count"를 "LM Count"로 바꾸고 "Curve" 컬럼 추가.
- 저장(Save/Next/Previous) 시 오브젝트 목록을 갱신하되 선택 유지.
- 리사이즈 후에도 오브젝트 미리보기 오버레이가 모서리에 스냅된 채 유지되도록 수정.
